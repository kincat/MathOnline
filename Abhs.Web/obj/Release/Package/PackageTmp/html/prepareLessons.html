学习内容

0是正数和负数的分界. 0℃是一个确定的温度，海拔0m表示海平面的平均高度，0的意义已不仅表示“没有”
在正数前加上符号“-”（负）的数叫做负数（小于0的数）.
正数大于0，0大于负数，正数大于负数 正整数、0、负整数，统称整数。 正分数、负分数统称为分数。
整数和分数统称为有理数（凡能写成形式q/p的数,p,q是整数且p≠0）
